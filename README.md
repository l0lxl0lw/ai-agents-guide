# AI Agents Guide

Beginner-friendly guides for setting up AI agent development tools. No prior coding experience required.

This repository walks you through installing the tools you need and creating your first AI agent project, step by step.

## Prerequisites

- **A computer**: Mac is recommended, but any modern computer (Windows/Linux) works.
- **An internet connection**: You'll need to download tools and sign up for accounts.
- **Curiosity**: That's it. We'll handle the rest together.
